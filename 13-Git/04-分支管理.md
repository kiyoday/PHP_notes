### 创建与合并

创建`dev`分支，并切换到`dev`分支：

```bash
$ git checkout -b dev

#以下两条命令
$ git branch dev
$ git checkout dev
```

`git branch`命令查看当前分支：

```bash
$ git branch
```

把`dev`分支合并到`master`分支上：

```bash
$ git checkout master
$ git merge dev
```

#### 新的`git switch`命令

创建并切换到新的`dev`分支，可以使用：

```bash
$ git switch -c dev
```

直接切换到已有的`master`分支，可以使用：

```bash
$ git switch master
```
