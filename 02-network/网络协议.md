## OSI七层模型 

### OSI七层模型 物联网淑慧试用

|        | 模型         | 功能                                                         |
| ------ | ------------ | ------------------------------------------------------------ |
| 第一层 | 物理层       | 建立、维护、断开物理连接                                     |
| 第二层 | 数据量链路层 | 建立逻辑连接、进行硬件地址寻址、差错校验等功能               |
| 第三层 | 网络层       | 进行逻辑地址寻址，实现不同网路之间的路径选择                 |
| 第四次 | 传输层       | 定义传输数据的协议端口号，以及流控和差错校验<br /> 协议有TCP、UDP，数据包一旦离开网卡即进入网络传输层 |
| 第五层 | 会话层       | 建立、管理、终止会话                                         |
| 第六次 | 表示层       | 数据的表示、安全、压缩                                       |
| 第七层 | 应用层       | 网络服务与最终用户的一个接口                                 |
